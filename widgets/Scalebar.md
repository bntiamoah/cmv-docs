# Scalebar

### scalebar
See [Class : Scalebar](https://developers.arcgis.com/javascript/jsapi/scalebar-amd.html) for parameters
#### Example Config Object
``` javascript
scalebar: {
    include: true,
    id: 'scalebar',
    type: 'map',
    path: 'esri/dijit/Scalebar',
    options: {
        map: true,
        attachTo: 'bottom-left',
        scalebarStyle: 'line',
        scalebarUnit: 'dual'
    }
},
```
