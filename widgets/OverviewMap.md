# Overview Map

#### Example Config Object
``` javascript
overviewMap: {
    include: true,
    id: 'overviewMap',
    type: 'map',
    path: 'esri/dijit/OverviewMap',
    options: {
        map: true,
               // location overview map will appear on Map object 
        attachTo: 'bottom-right',
        color: '#0000CC',
        height: 100,
        width: 125,
        opacity: 0.30,
        visible: false
    }
},
```
